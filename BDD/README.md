# Optimizaciones y Notas Técnicas - MySatcom

Este repositorio contiene el respaldo de scripts, procedimientos almacenados y documentación técnica relacionada con la optimización de la base de datos MySatcom.

## Contenido

### 📂 BDD/SPS (Stored Procedures)
- **Consultas Masivas Portal:** Mejoras en el rendimiento de consultas de comprobantes.
- **Info Reportes:** Procesos de población de datos para tableros de control.
- **LOGs Monitoreo:** Herramientas de auditoría y trazabilidad de ejecuciones.
- **No Autorizados 2026:** Lógica para la detección y reporte de comprobantes no autorizados.

### 📂 BDD/Estructuras
- Definiciones de tablas críticas y manual técnico de la base de datos.

### 📂 BDD/EjemplosTramas
- Conjunto de archivos XML (facturas, notas de crédito, retenciones) para pruebas de integración y procesamiento.

## Propósito
El objetivo de este repositorio es mantener un historial limpio y portable de las optimizaciones realizadas en el entorno de Satcom, independiente del código fuente de la aplicación principal.
